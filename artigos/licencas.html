Licenças de software livre

Apache 2.0

É uma licença de software livre, compatível com a versão do GNU GPL, não e compatível com versão GPL 2, já que possui alguns requisitos que não tem na versão GPL, inclui certas disposições de rescisão de patentes e indenização, a provisão da rescisão das patentes é uma coisa muito boa, e é por isso que é recomendado este tipo de licença para programas com licenças permissivas ruins.

FreeBSD

É uma licença original de bsd, e possui a cláusula de publicidade removida, pode ser chamada de licença BSD 2 de clausulas, é uma licença de software livre não copiável e permissiva, é compatível com o GPL GNU.

Universal Permissive License

Trata-se de uma licença de software livre não copiável, também é permissiva, é compatível com GPL GNU. Ela fornece a capacidade de licenciar patentes juntamente com o trabalho de software, mas, ainda é recomendado a licença apache 2.0, para se evitar plagio de patente ao optar por colocar seu trabalho em uma licença.

GNU General Public License 3

É a versão mais recente do GPL GNU, é uma licença copyleft, e é recomendada para a maioria dos pacotes de software. A gplv3 não é compatível com a gplv2 por si só, mas, no entanto, a maioria dos softwares lançados sob gplv2 permite que você use os termos das versões posteriores do gpl também, quando é o caso, você pode usar o código gplv3 para se fazer a combinação desejada.

Educational Community License 2.0

É uma licença de software livre, é compatível com GPLv3, é baseado na licença apache 2.0, o escopo da licença de patente mudou de modo que, quando o funcionário de uma organização trabalha em um projeto, a organização não é obrigada a liberar/licenciar todas as patentes aos destinatários, esta licença de patente e a clausula de indenização na seção 9, tornam esta licença incompatível com a GPLv2.

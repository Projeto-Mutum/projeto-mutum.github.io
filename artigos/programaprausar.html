Programas para você utilizar

Linux

Linux Apesar de ser um kernel, que pode ser definido como a "coluna vertebral" de um sistema operacional. Por ser de código aberto, o Linux possibilita uma diversa gama de opções para você utilizar como seu sistema operacional sendo algumas delas: Debian, Arch Linux e Slackware

Terminal

Alacritty Este emulador de terminal tem ganhado muita fama nos últimos anos, principalmente por conta de ter acelaração com placa de vídeo, o que torna diversos processos como a compilação ocorram em menos tempo.

Editor de texto

Essas ferramentas auxiliam na configuração do seu sistema, possibilitando com que você edite arquivos do diretório /etc, por exemplo. O Vim pode ser executado diretamente em seu terminal, possui uma interface simples, e é muito poderoso, sendo altamente customizável, aceitando até plugins. Já o Emacs é uma ferramenta um pouco mais pesada, passando de 1 milhão de linhas em seu código fonte, e possui até uma linguagem própria o elisp, se você tiver paciência para aprender a utilizá-lo, estará utilizando uma ótima ferramenta que consegue fazer muitas coisas.

Navegadores

O navegador, é onde passamos mais tempo, seja para entretenimento ou pesquisas, portanto é importante escolhermos um navegador de qualidade que respeite nossa privacidade e funcione de maneira eficiente.

Firefox

Firefox Sendo uma das ferramentas mais antigas que possui seu código aberto, o Firefox, respeita sua privacidade, e conta com diversas proteções relacionadas a rastreadores que são incluidos nas páginas que acessamos, inclusive, existem modificações do navegador que o deixam mais seguro ainda, por exemplo a modificação do arkenfox.

Brave

O Brave é baseado no chrome, porém, não possui a parte invasiva. Ele conta com um bloqueador de anúncios embutido, gera relatórios de privacidade, possui um sistema próprio de buscas e inclusive, possui sua própria criptomoeda, você pode ganha-las por apenas utilizar o navegador.

Gerenciador de arquivos

Ranger

Ranger nada mais é que um gestor de arquivos numa linha de comando, inteiro escrito em python, que possui todas as funções mais usadas, sendo umas delas o Nautilus, no Ubunu, tirando que possui uma velocidade de execução muito rápida e eficaz.

Reprodutor de vídeo

MPV

MPV Possui o código aberto para usuários de Linux, Windows e outros sistemas operacionais, tendo uma interface minimalista, simples é de fácil compreensão, tem todos os recursos padrões de qualquer outro editor de vídeos, podendo ser instalando até em máquinas mais fracas.
